.. _grids:

Grids
================================================

Meso-NH use a C-grid in the Arakawa convention :cite:`arakawa_computational_1977`, both on the horizontal and on the vertical. This grid is shown on the following figure with:

* 1 : mass points

* 2 : u points

* 3 : v points

* 4 : w points

* 5 : vertical vorticity point

* 6 : vorticity components along y

* 7 : vorticity components along x

.. figure:: grids.png
   :align: center
   :alt: Meso-NH's grids

   Meso-NH's grids.
