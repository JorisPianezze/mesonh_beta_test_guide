* :code:`LVISC_UVW` : viscosity for the momentum

* :code:`LVISC_SV` : viscosity for the scalar tracer

* :code:`LVISC_R` : viscosity for the moisture
 
* :code:`XMU_V` : Molecular (cinematic) viscosity

* :code:`XPRANDTL` : Prandtl number
